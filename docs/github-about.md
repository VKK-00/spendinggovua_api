# GitHub About

## Repository description

Browser-backed FastAPI service and UI for extracting reports from spending.gov.ua by EDRPOU, period, and report form.

## Short project pitch

This project provides a practical API and web interface for working with public spending reports from `spending.gov.ua`. It searches reports by one or many EDRPOU codes, filters them by form and period, exports results to ZIP, and can render individual reports as human-readable HTML or PDF.
